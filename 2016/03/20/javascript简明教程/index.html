Function @

  • 没有重载（深入理解）

    因为函数名为指针，相同的函数名指向的为同一个函数

    • 函数声明

      可以无视代码顺序

    • 函数表达式

      必须在表达式之后才可引用

    可以作为参数传递或返回，实际上就是个引用
